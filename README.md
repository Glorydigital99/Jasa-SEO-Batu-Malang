
# Glory Digital Agency - Project Setup

Proyek ini menggunakan React, TypeScript, dan Vite.

## Cara Menjalankan di VS Code

1. **Instalasi Node.js**: Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/).
2. **Download/Clone**: Letakkan semua file proyek di dalam satu folder.
3. **Buka Terminal**: Di VS Code, buka terminal (Ctrl+`) dan jalankan:
   ```bash
   npm install
   ```
4. **Setup Environment**:
   - Copy file `.env.example` menjadi `.env`.
   - Masukkan API Key Gemini Anda pada `VITE_API_KEY`.
5. **Jalankan Project**:
   ```bash
   npm run dev
   ```
6. **Akses**: Buka `http://localhost:3000` di browser Anda.

## Struktur File
Proyek ini menggunakan struktur flat di root untuk kemudahan akses komponen.
